+) Texture
+) Background
+) Event bàn phím
+) Event chuột
+) Animation
+) Xử lý va chạm
+) Score(có tính điểm)
+) Lưu kỷ lục điểm cao
+) Sound
+) Background music
+) Font
+) Menu
+) Pause/Resume
